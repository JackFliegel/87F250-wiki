# 1987 Ford F-250 Knowledge Base

This vault contains immutable raw source material in `raw/` and synthesized markdown knowledge in `wiki/`.

Rules:
- Never edit files in `raw/`
- Store original giant files in `raw/archive/`
- Store working manuals and split source files in `raw/manuals/`
- Store synthesized knowledge only in `wiki/`
- Every wiki page should cite its source files
- Log meaningful wiki changes in `wiki/log.md`
